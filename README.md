# LuaTools Patcher для темы Fluenty (Millennium)

Удобный PowerShell-патчер, который автоматически русифицирует боковую панель темы Fluenty в Steam, интегрирует кнопку быстрого доступа к **LuaTools**, генерирует необходимые стили (Quick CSS), а также может **автоматически устанавливать** свежую версию темы напрямую из репозитория GitHub!

**Автор интеграции:** Le Maxime ([t.me/lemaxime](https://t.me/lemaxime))  
**Версия патчера:** 1.2  
**Поддерживаемая версия Fluenty:** 1.14.0 (с автоматической проверкой версии в `skin.json`)  

---

## Что делает патчер?

1. **Два режима работы (на выбор пользователя):**
   * **Пропатчить установленную тему:** Патчер автоматически находит вашу текущую тему Fluenty и применяет к ней модификации.
   * **Скачать, установить и пропатчить тему:** Патчер автоматически скачивает архив темы Fluenty из этого репозитория, устанавливает её в папку скинов Steam и применяет все патчи.
2. **Проверка версии темы:**
   * Перед патчингом автоматически считывает версию Fluenty из `skin.json` и сопоставляет её с поддерживаемой версией (`1.14.0`).
   * Если версии не совпадают, патчер выводит предупреждение и предлагает выбор: продолжить или отменить операцию.
3. **Динамический перевод боковой панели (JS-инъекции):**
   * Переводит все разделы бокового меню на русский язык (Магазин, Библиотека, Коллекции, Сообщество, Торговая площадка, Активность, Загрузки, Друзья, Настройки).
   * Автоматически вживляет кнопку **LuaTools** в боковое меню и связывает её с логикой отображения LuaTools.
4. **Адаптация страницы игры (`viewGame.js`):**
   * Корректно оборачивает заголовок игры для интеграции кнопок LuaTools, SteamDB и Online-Fix без конфликтов.
   * Добавляет плавный скролл при клике на отзывы.
5. **Генерация Quick CSS:**
   * Создает файл стилей `quick-css-code.css` в папке запуска.
   * **Автоматически копирует** эти стили в буфер обмена Windows для быстрой вставки.

## Инструкция по использованию

### Вариант 1: Быстрый запуск в одну строку (Рекомендуется)

Откройте PowerShell и выполните следующую команду (скрипт скачается и запустится автоматически без сохранения на диск):

```powershell
irm https://raw.githubusercontent.com/DarkAssassinUA/fluenty-luatools-patcher/main/patch_fluenty.ps1 | iex
```

*Примечание: Если тема устанавливается/патчится в системную папку Steam (`Program Files (x86)`), скрипт автоматически запросит права администратора (UAC) во всплывающем окне.*

---

### Вариант 2: Запуск локального файла

1. Загрузите файл [**`patch_fluenty.ps1`**](https://raw.githubusercontent.com/DarkAssassinUA/fluenty-luatools-patcher/main/patch_fluenty.ps1) на ваш компьютер.
2. Нажмите по файлу правой кнопкой мыши и выберите **«Выполнить с помощью PowerShell»** (Run with PowerShell).

---

## Что делать после работы скрипта?

1. Откройте настройки Millennium в клиенте Steam.
2. Перейдите во вкладку **«Quick CSS»**.
3. Вставьте скопированный код из буфера обмена (нажмите **Ctrl+V**).
4. Нажмите кнопку сохранения стилей и перезапустите Steam (или перезагрузите скин).

---

## Безопасность при обновлениях темы

Поскольку Millennium полностью затирает файлы темы при её обновлении:
* Все ваши настройки стилей в **Quick CSS** останутся нетронутыми (Millennium хранит их отдельно).
* После обновления темы вам потребуется всего лишь **запустить команду в PowerShell ещё раз**, чтобы вернуть перевод и кнопку LuaTools в JS-файлы темы.
